# Foco Salvaje — Cristian Cicconofri

Sitio web para venta de fotos del Campeonato de Pesca 2025.

## Estructura

```
foco-salvaje/
├── index.html        # Página principal
├── css/
│   └── styles.css    # Todos los estilos
├── js/
│   └── app.js        # Lógica: galería, carrito, navegación
└── assets/           # Acá van las fotos reales
```

## Próximos pasos

1. Reemplazar las fotos de ejemplo en `app.js` con las fotos reales
2. Conectar MercadoPago con las credenciales de la cuenta
3. Subir a Netlify o Vercel (gratis)

## Cómo subir a Netlify

1. Ir a https://netlify.com
2. Arrastrar la carpeta `foco-salvaje` al panel
3. ¡Listo! La página queda online en segundos
